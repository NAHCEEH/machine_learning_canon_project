# machine_learning_canon_project

# 캐논 프로젝트: 실시간 타겟 화면 순차 검출 시스템
>**머신러닝 3조 팀 프로젝트** 

본 프로젝트는 핸디형 카메라로 촬영된 산업 현장 영상에서 특정 Target 화면 4종을 노이즈에 구애받지 않고 안정적으로 순차 검출하는 시스템 구축을 목표

---

##  프로젝트 개요
* **목적:** 실제 산업 현장의 조명 반사, 흔들림 등 노이즈 환경에서도 특정 화면 등장 여부를 빠르게 판단
* **핵심 기능:** * 이미지/영상/실시간 카메라 입력 기반 Target 1~4 판별
    *상태 머신 기반의 순차 판단 로직 (Target 1 → 2 → 3 → 4) 
    * 비전문가 관리자를 위한 직관적 인터페이스 제공 

---

##  주요 해결 전략 
1. **강인한 AI 모델링:** Object Detection 및 Anomaly Detection 기법을 활용하여 유사 화면 오탐지 최소화 
2. **신뢰도 기반 판단:** 연속 프레임 일치 여부 및 Confidence Score를 활용한 확정 로직 
3. **E2E 시스템 통합:** 단순 모델을 넘어 인식-판단-표시가 실시간으로 이어지는 완성형 프로그램 구현

---

## 팀원 역할 
| 파트 | 이름 | 주요 역할 |
| :--- | :---: | :--- |
| **AI Researcher** | 팀원 A, B, C | 모델 학습, 데이터 증강, 로직 설계|
| **Backend** | 팀원 D | 실시간 영상 스트리밍 파이프라인 및 데이터 로그 저장  |
| **Frontend** | 팀원 E | 테스트/실시간 모드 대시보드 및 비전문가용 UI 구현 |

---

##  평가 지표
* **분류 성능:** Accuracy, Precision, Recall, F1-score 
* **순차 검출:** Sequence Accuracy (Target 1~4의 올바른 순서 검증) 
* **실시간성:** 초당 처리 프레임 수 (FPS) 및 지연 시간 최소화
